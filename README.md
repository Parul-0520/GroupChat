# Group Chat Application
A multi-client chat app built in Java using Swing for the UI and Sockets for real-time communication.

## Features
- Real-time group messaging across multiple clients
- Custom Swing UI resembling a modern chat interface
- Inline colored emoji support (custom-built, no external libraries)
- Scrollable message history with auto-scroll
- Enter-to-send and click-to-send support

## How to run
1. Run `Server.java` first 
2. Run `UserOne.java`, `UserTwo.java`, `UserThird.java` — each opens a separate chat window
3. Type a message and press Enter or click Send

## Tech used
Java, Swing, Java Sockets, Multithreading
